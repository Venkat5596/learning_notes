git init  # to intial git


Start
  │
  ├── Initialize Repository → `git init`
  │
  ├── Check Status → `git status`
  │
  ├── Add Files to Staging → `git add <filename>` OR `git add .`
  │
  ├── Commit Changes → `git commit -m "Commit message"`
  │
  ├── Set Remote Repository → `git remote add origin <repository-url>`
  │
  ├── Push Changes → `git push origin <branch-name>`
  │
  ├── Pull Latest Changes → `git pull origin <branch-name>`
  │
  ├── Branch Management
  │       ├── Create New Branch → `git branch <branch-name>`
  │       ├── Switch Branch → `git checkout <branch-name>`
  │       ├── Merge Branch → `git merge <branch-name>`
  │
  ├── Check Logs → `git log`
  │
  ├── Clone Repository → `git clone <repository-url>`
  │
  ├── Resolve Merge Conflicts → `git diff` & `git mergetool`
  │
  ├── Revert Changes → `git reset --hard <commit-hash>` OR `git revert <commit-hash>`
  │
  └── End
